# PLANTICA - Bitki Bakım Rehberi

Evde veya bahçede yetiştirilen bitkilerin sağlıklı gelişimini desteklemek için geliştirilmiş bir bitki bakım rehberi uygulamasıdır.

## Özellikler
- Bitki bilgileri görüntüleme
- Bakım önerileri
- Sulama bilgileri
- Kullanıcı dostu arayüz

## Kullanılan Teknolojiler
- Java
- Eclipse IDE
